# BJ_counter

## Usage
[BJ counter](https://bj-counter-1b168.firebaseapp.com/)

1. Select deck size
1. Count with Buttons
    + Instead, Use Keyboard 

    | Key | Point | Number |
    | --- | ----- | ------ |
    | A   | +1    | 2-7    |
    | S   | 0     | 8-9    |
    | D   | -1    | 10-A   |

1. For Reset, Select deck size again OR Leave No cards OR Press `Enter Key`
